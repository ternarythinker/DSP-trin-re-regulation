## How to Contribute

### 1. Theoretical Refinements

- Derive new consequences from ρ_t equations
- Find solutions to field equations
- Extend to higher dimensions
- Quantum gravity applications

**Submit:** Pull request with derivation + code

### 2. Experimental Validation

- Reproduce measurements using ρ_t formulas
- Compare predictions to published data
- Identify new testable predictions
- Design experiments

**Submit:** Pull request with data analysis + code

### 3. Code Optimization

- Faster simulations
- Better numerical methods
- Parallel implementations
- Quantum computing approaches

**Submit:** Pull request with benchmarks

### 4. Applications

- Robotics (ρ_t control systems)
- Medicine (thermal models)
- Materials (ternary structures)
- AI (consciousness models)

**Submit:** New directory with examples

### 5. Documentation

- Tutorials
- Visualizations
- Explainer videos
- Pedagogical materials

**Submit:** Pull request to `/docs`

## Submission Guidelines

### Before Submitting

- [ ] Code runs without errors
- [ ] Matches existing code style
- [ ] Includes docstrings/comments
- [ ] Tests pass (if applicable)
- [ ] No duplicate work


## Physics Standards

### Derivations

- Start from first principles (ρ_t field equations)
- Show all steps
- Cite relevant literature
- Verify against experiment

### Simulations

- Use SI units throughout
- Include physical constants
- Document approximations
- Compare to known results

### Experiments

- Use published measurements
- Include error bars
- Calculate deviations rigorously
- Blind comparison (predict first!)
